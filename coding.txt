1.Absract:
  The Password Generation System is a software application designed to generate strong and secure passwords automatically. 
  Strong passwords are essential for protecting user accounts from unauthorized access. This system creates random passwords 
  using a combination of uppercase letters, lowercase letters, numbers, and special characters. The generated passwords are 
  difficult to guess and improve overall security. The project can be implemented using programming languages

2.Itroduction:
 the rapid growth of digital systems, securing user accounts has become very important. Weak passwords can be easily hacked 
 using brute force or dictionary attacks. Therefore, generating strong and random passwords is necessary. The Password Generation 
System helps users create secure passwords instantly without manual effort

3.Methodology
 *The system follows these steps:
 *User enters the desired password length
 *System selects random characters from:
 *Uppercase letters
 *Lowercase letters
 *Numbers
 *Special symbols
 *System combines the characters randomly
 *System displays the generated password

4.coding:
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

/* Function to generate password */
void generatePassword(int length)
{
    /* Character set for password */
    char upper[] = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
    char lower[] = "abcdefghijklmnopqrstuvwxyz";
    char numbers[] = "0123456789";
    char symbols[] = "!@#$%^&*";

    /* Combined character array */
    char all[100];

    int i = 0;
    int j = 0;

    /* Copy uppercase letters */
    while(upper[i] != '\0')
    {
        all[j] = upper[i];
        i++;
        j++;
    }

    i = 0;

    /* Copy lowercase letters */
    while(lower[i] != '\0')
    {
        all[j] = lower[i];
        i++;
        j++;
    }

    i = 0;

    /* Copy numbers */
    while(numbers[i] != '\0')
    {
        all[j] = numbers[i];
        i++;
        j++;
    }

    i = 0;

    /* Copy symbols */
    while(symbols[i] != '\0')
    {
        all[j] = symbols[i];
        i++;
        j++;
    }

    int totalCharacters = j;

    printf("\nGenerated Password: ");

    /* Generate password using loop */
    for(i = 0; i < length; i++)
    {
        int randomIndex;

        randomIndex = rand() % totalCharacters;

        printf("%c", all[randomIndex]);
    }

    printf("\n");
}

/* Main function */
int main()
{
    int length;

    /* Initialize random seed */
    srand(time(0));

    printf("====================================\n");
    printf("     PASSWORD GENERATION SYSTEM     \n");
    printf("====================================\n");

    /* Get password length from user */
    printf("Enter desired password length: ");
    scanf("%d", &length);

    /* Check valid length */
    if(length <= 0)
    {
        printf("Invalid length. Please enter positive number.\n");
    }
    else
    {
        generatePassword(length);
    }

    printf("\nProgram Finished Successfully.\n");

    return 0;
}
